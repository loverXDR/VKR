О проекте

Проект "Система рекомендаций тематических ресурсов" разработан в рамках выпускной квалификационной работы.

Основная цель проекта - создание интеллектуальной системы поиска и рекомендаций тематических ресурсов, использующей современные алгоритмы машинного обучения и векторные представления текстов.

Архитектура системы

Основные компоненты:
  • Парсеры данных: Модули для сбора информации с тематических сайтов
  • Векторная база данных (Qdrant): Хранение ресурсов и их векторных представлений
  • Модель Word2Vec: Создание семантически значимых векторных представлений текстов
  • Поисковая система: Семантический поиск на основе векторных представлений
  • Веб-интерфейс: Взаимодействие пользователя с системой

Принцип работы

Система работает по следующему принципу:

  1. Сбор данных: Парсеры собирают информацию с тематических сайтов и сохраняют её в базу данных.
  2. Обработка данных: Каждый ресурс обрабатывается с помощью естественно-языковых моделей.
  3. Векторизация: Для каждого ресурса создается векторное представление с помощью модели Word2Vec.
  4. Индексация: Векторные представления индексируются в базе данных Qdrant для быстрого поиска.
  5. Поиск: При запросе пользователя текст запроса преобразуется в вектор, после чего в базе данных ищутся ресурсы с наиболее похожими векторами.
  6. Ранжирование: Найденные ресурсы ранжируются по степени семантической близости к запросу.
  7. Отображение: Наиболее релевантные результаты показываются пользователю.

Перспективы развития

В будущем планируется расширение функциональности системы:

  • Добавление новых источников данных
  • Улучшение алгоритмов семантического анализа
  • Внедрение персонализированных рекомендаций на основе истории поиска пользователя
  • Реализация системы обратной связи для улучшения результатов поиска
  • Оптимизация производительности для работы с большими объемами данных
